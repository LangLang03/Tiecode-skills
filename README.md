# tiecode-tlang

Tiecode `.t` 语法与生成规范 Skill。

## 目录
- `SKILL.md`：Skill 触发规则与执行流程
- `agents/openai.yaml`：Skill UI 元数据
- `references/`：语法、实例、规范、证据、Schema
- `references/indexes/`：API 与结构化索引 JSON

总览入口：
- `references/overview.md`
