# 错误处理与调试

## 1. 容错处理

### 1.1 基本容错结构

```t
方法 安全执行()
	容错处理()
		可能出错的代码()
		返回 真
	结束容错()
	返回 假
结束 方法
```

规则：
- `容错处理()` 开始容错块
- `结束容错()` 结束容错块
- 容错块内的代码出错时跳到 `结束容错()` 后继续执行
- 适合文件操作、网络请求、类型转换等可能出错的操作

### 1.2 带错误信息的容错

```t
方法 读取文件(路径 : 文本) : 文本
	开始俘获异常()
		变量 内容 : 文本 = 文件.读取文本(路径)
		返回 内容
	俘获所有异常()
		变量 错误 : 异常 = 取俘获异常()
		弹出提示("读取失败: " + 错误.取消息())
		返回 ""
	结束俘获异常()
	返回 ""
结束 方法
```

规则：
- `开始俘获异常()` 开始异常捕获块
- `俘获所有异常()` 捕获所有异常
- `取俘获异常()` 获取捕获的异常对象
- `结束俘获异常()` 结束异常捕获块

## 2. 调试输出

### 2.1 基本调试

```t
方法 调试示例()
	调试输出("开始执行")
	变量 结果 : 整数 = 计算()
	调试输出("计算结果: " + 结果.到文本())
结束 方法
```

规则：
- `调试输出(内容)` 输出到日志
- 支持文本、数字、对象等类型
- 仅在调试模式有效，发布时自动移除

### 2.2 格式化调试

```t
方法 格式化调试示例()
	变量 名称 : 文本 = "测试"
	变量 数值 : 整数 = 100
	调试输出2("名称: %s, 数值: %d", {名称, 数值})
结束 方法
```

规则：
- `调试输出2(格式, 参数数组)` 格式化输出
- `%s` 文本占位符，`%d` 整数占位符
- 参数使用数组传递

## 3. 常见错误处理模式

### 3.1 空值检查

```t
方法 安全访问(对象 : 对象?)
	如果 对象 == 空
		返回
	结束 如果
	// 安全访问对象成员
结束 方法
```

规则：
- `类型?` 表示可空类型
- 使用前必须检查 `== 空`
- 避免空指针异常

### 3.2 数组边界检查

```t
方法 安全访问数组(数组 : 文本[], 索引 : 整数) : 文本
	如果 索引 < 0 或 索引 >= 取数组长度(数组)
		返回 ""
	结束 如果
	返回 数组[索引]
结束 方法
```

规则：
- 访问数组前检查索引范围
- `取数组长度(数组)` 获取数组长度
- 避免数组越界异常
