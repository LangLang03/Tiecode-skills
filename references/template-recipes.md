# 模板配方（高复用实例）

## 1. 主窗口页面模板（推荐默认）
```t
类 启动窗口 : 窗口
	@布局配置([[根布局=真,宽度=-1,高度=-1]])
	变量 根布局 : 相对布局

	事件 启动窗口:创建完毕()
		本对象.显示标题栏 = 假
	结束 事件
结束 类
```

规则：
- 页面继承 `窗口`。
- 不写 `包名`。
- 不写 `@code` 构造器样板。

## 1.1 多组件布局声明（连续块）
```t
类 测试页面 : 窗口

	@布局配置([[根布局=真,宽度=-1,高度=-1]])
	变量 根布局 : 相对布局
	@布局配置([[父布局=根布局,宽度=-2,高度=-2,ID=1,内容="标题",字体大小=24,@位于布局中间=真]])
	变量 标题文本 : 文本框
	@布局配置([[父布局=根布局,宽度=-2,高度=-2,ID=2,内容="提示文本",字体大小=18,@位于某组件之下=1,@位于布局中间=真]])
	变量 分数文本 : 文本框
	@布局配置([[父布局=根布局,宽度=-2,高度=-2,ID=3,内容="等待结果",字体大小=20,@位于某组件之下=2,@位于布局中间=真]])
	变量 结果文本 : 文本框
结束 类
```

规则：
- `类 ...` 后保留一个空行，再开始布局声明块。
- 每个 `@布局配置` 与其后 `变量` 必须紧邻。
- 各声明对之间不插空行；相邻 `@布局配置` 定义之间不允许出现空行。
- 布局声明块结束后的空行不作约束。
- 布局值支持多类型：文本（带引号）、数字、逻辑型（`真/假`），并与目标属性/方法签名匹配。
- 颜色布局值（如 `背景颜色`）必须使用十进制有符号整数，禁止 `0x/0X` 十六进制写法。
- 文本默认不引入无意空格：例如 `猜正面`、`猜反面`、`总次数:0|胜利:0|失败:0`。

## 2. 工具类（静态方法）
```t
@全局类
@禁止创建对象
类 文本工具
	@静态
	方法 为空(值 : 文本) : 逻辑型
		code return #值 == null || #值.isEmpty();
	结束 方法
结束 类
```

## 4. 读写属性组件
```t
类 标题栏
	属性写 标题(值 : 文本)
		code getView().setTitle(#值);
	结束 属性

	属性读 标题() : 文本
		code return getView().getTitle().toString();
	结束 属性
结束 类
```
